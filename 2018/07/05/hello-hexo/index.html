Hexo的简单使用

常用命令

npm install hexo-cli -g
hexo init demo 或者 hexo init . 在当前目录创建hexo项目
npm install
hexo g 等同于 hexo generate，生成html文件
hexo s 起本地服务可以预览
hexo d 发布到指定的github
